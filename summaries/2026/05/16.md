**Score:** 558 | **Comments:** 354 | **ID:** 48143880

> **Article:** A security researcher discovered that Mullvad VPN exit IPs are surprisingly identifying and can be used to fingerprint users. The exit IP assigned to users is not randomized on each connection but is deterministically picked based on the user's WireGuard key, which only rotates every 1-30 days. This correlation between WireGuard keys and exit IPs means that different accounts using different Mullvad servers can still be linked to the same person if their IPs fall within overlapping float ranges, potentially allowing for 99% confidence in identifying sockpuppets or the same user across different connections.
>
> **Discussion:** Mullvad's co-CEO acknowledged the findings, stating some behavior is intentional while some is not, and they're already testing a patch. A key technical explanation from Obscura's CEO clarified that WireGuard is connectionless by design, and frequent IP rotation would disrupt transport-layer connections and trigger application-level security measures like CAPTCHAs and logouts. The discussion also featured a heated debate about VPN trustworthiness, with some commenters arguing VPNs are "snake oil" that simply transfer trust from ISPs to commercial companies, while others countered that VPNs can be chosen based on jurisdiction and court-proven no-log policies.
